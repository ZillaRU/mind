# 慢 mind — 使用文档


> AI 在飞速流淌，你在慢慢生活

## 🌿 这是什么

「慢 mind」是一个帮助你在 AI 时代找回内心平静的 Web 应用。

它的核心理念来自一个真实的体验：**当 AI 在屏幕上飞速生成代码时，你在旁边慢慢练琴，反而感受到了一种前所未有的专注和宁静。**

这不是一个生产力工具，没有 streak、没有排行榜、没有推送通知。它只是一个安静的陪伴空间。

---

## 🎯 核心功能

### 1. Vibe Screen（氛围屏幕）
- 屏幕背景上，AI 在安静地「写代码」——真实的、有意义的代码片段
- 代码以打字机效果逐行生长，偶尔穿插文学名句
- 不是给你看的，是给你「感受到」的背景氛围
- 支持 TypeScript、Python、Rust、Go、Haskell、JavaScript 六种语言

### 2. 慢活动引导
- 10 种内置「不能被 AI 加速」的活动：练琴、做饭、拉伸、散步、手冲咖啡、随手画、读纸质书、写手账、照顾植物、只是呼吸
- 每个活动都有温暖的引导语
- 按类别筛选：动手、身体、感官、创造

### 3. ✨ 自定义活动（新）
- 点击活动列表底部的 **「+ 添加你的活动」** 卡片
- 填写名称、选一个图标、写描述和引导语
- 自定义活动会保存在浏览器本地，刷新不丢失
- 可以随时编辑或删除自定义活动（点击右上角 ··· 菜单）
- 每个人进入心流的方式不同，创造你自己的慢活动清单

### 4. ✨ 随机灵感（新）
- 点击顶部栏的 **「✨ 灵感」** 按钮
- 随机展示一首经典音乐、一首诗、或一幅画
- **音乐**：8 首公共领域经典（萨蒂、德彪西、肖邦、巴赫、贝多芬、维瓦尔第、舒伯特），附带搜索提示
- **诗歌**：10 首经典（陶渊明、张若虚、李白、王维、苏轼、李煜、Frost、Kipling、Wordsworth），中英双语
- **画作**：7 幅公共领域名画（梵高、莫奈、北斋、蒙克、修拉），来自 Wikimedia Commons
- 每条内容都配有产品气质的解读文字
- 点击「换一首/换一幅」随机切换
- 所有内容均为公共领域，无版权风险

### 5. 无压计时器
- 没有倒计时，没有闹钟，没有目标
- 只是一个温柔的沙漏，记录你「慢」了多久
- 呼吸光环动画，与你的节奏同步

### 6. 手记
- 活动结束后，可选地写几句感受
- 纯本地存储（localStorage），不上传、不分享、不打分
- 可以回顾过去的「慢时光」

### 7. 氛围音
- 6 种程序化生成的环境音：静默、雨声、炉火、咖啡馆、风声、海浪
- 使用 Web Audio API 实时合成，无需下载音频文件
- 平滑淡入淡出切换

### 8. 粒子背景
- Canvas 绘制的漂浮光点
- 紫蓝色调，缓慢飘动
- 为整个页面增添梦幻氛围

## 🚀 快速开始

### 本地开发

```bash
# 进入项目目录
cd mind

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 部署

将 `dist/` 目录部署到任何静态托管服务即可：

- **Netlify**：直接拖拽 `dist/` 到 https://app.netlify.com/drop
- **Vercel**：`npx vercel --prod`
- **GitHub Pages**：推送到仓库，开启 Pages

## 🎨 设计理念

### 视觉
- **深色主题**：午夜蓝底色，减少视觉刺激
- **紫蓝色调**：aurora（极光紫）作为主色调，营造梦幻感
- **大量留白**：让内容呼吸
- **慢速动画**：所有过渡都是 500ms-1000ms，不急
- **渐变遮罩**：代码从底部渐隐，像是从深渊中生长

### 听觉
- **程序化合成**：所有声音都是 Web Audio API 实时生成的
  - 雨声 = 棕噪音（低频）+ 粉噪音（中频）+ 白噪音（高频滴水）
  - 炉火 = 棕噪音噼啪 + 白噪音 + 60Hz 正弦波低鸣
  - 海浪 = 棕噪音 + LFO 调制（8秒周期模拟潮汐）
  - 风声 = 棕噪音 + LFO 调制（模拟阵风）
- **平滑过渡**：切换声音时 1 秒淡出 + 2 秒淡入
- **零外部依赖**：不需要下载任何音频文件

### 交互
- **零摩擦**：打开就能用，不需要注册登录
- **无压力**：没有倒计时焦虑，没有完成感压力
- **个性化**：自定义活动让每个人找到自己的慢方式
- **灵感随机**：不经意间遇见一首诗、一幅画、一首曲子
- **可选记录**：手记是可选的，跳过也完全 OK
- **反社交**：没有分享按钮，没有社交功能
