## 项目概览

**慢 mind** — 一个反提效、反焦虑的 Web 应用

- **核心理念**：没有倒计时，只有此刻；允许所有情绪，只要是真的在体内流淌着的
- **技术栈**：React + TypeScript + Vite + Tailwind CSS v4
- **部署**：GitHub Pages，地址 https://zillar.github.io/mind/

### 功能模块

- **5 套主题**：aurora（极光）、leaf（绿叶）、water（水波）、twilight（暮色）、ink（墨）
- **10 个活动**：拉伸、呼吸、手冲咖啡、随手画、散步、读纸质书、练琴、做饭、照顾植物、写手账
- **每个活动有专属体验**：
  - 拉伸：姿势循环 + ASCII art + 诗句引导
  - 呼吸：动态呼吸圆环 + 呼吸诗歌
  - 咖啡/做饭：步骤引导（无倒计时）
  - 画画/散步/读书/植物/手账：提示循环（诗句、引导语）
  - 练琴：可视化节拍器
- **其他**：粒子背景、环境音效、手账记录、灵感卡片

### 设计风格

- 萌、朦胧，像半梦半醒时听到的声音
- 允许 emo，不强行正向
- 强调"余地"——页面四周留白，元素之间有呼吸空间
- 内容随机性：打乱顺序、随机间隔、随机视觉变体
